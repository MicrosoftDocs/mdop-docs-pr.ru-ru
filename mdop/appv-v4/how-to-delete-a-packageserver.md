# Удаление пакета


С помощью описанной ниже процедуры можно удалить пакет, в том числе все версии пакета, из консоли управления сервером Application Virtualization. Это можно сделать для упрощения управления файлами на сервере, а также для того, чтобы удалить пакет и заменить его более стабильным.

**Важно!**  При удалении пакета появляется поле подтверждения, в котором будет выводится напоминание о том, что это действие удалит все версии. Сервер больше не сможет передавать приложение в поток.

 

**Удаление пакета**

1.  В левой области консоли управления сервером виртуализации приложений разверните раздел **пакеты**.

2.  В области дерева меню щелкните пакет, который нужно удалить, правой кнопкой мыши и выберите команду **Удалить**.

3.  Прочтите окно подтверждения и нажмите кнопку **Да** , чтобы завершить действие.
