# Изменение существующего пакета виртуальных приложений (App-V 4.6 с пакетом обновления 1 (SP1))


Чтобы изменить существующий виртуальный пакет приложения, выполните указанные ниже действия. Эти процедуры можно использовать в следующих случаях:

-   Обновите приложение, которое входит в существующий виртуальный пакет приложения. Для выполнения этой задачи используйте процедуру **"обновление приложения в существующем пакете приложения"** в этом документе.

-   Измените свойства существующего виртуального пакета приложения. Чтобы выполнить эту задачу, выполните действия, описанные в статье **как изменить свойства, связанные с существующим пакетом виртуальных приложений** , в этом документе.

-   Добавьте новое приложение в существующий виртуальный пакет приложения. Для выполнения этой задачи используйте процедуру **"Добавление нового приложения в существующий виртуальный пакет приложения"** в этом документе.

Для изменения виртуального пакета приложения необходимо установить секвенсор App-V. Дополнительные сведения об установке секвенсора App-V можно найти в разделе [Установка секвенсора (App-v 4,6 SP1)](how-to-install-the-sequencer---app-v-46-sp1-.md).

**Обновление приложения в существующем виртуальном пакете приложения**

1.  Чтобы запустить секвенсор App-v на компьютере, на котором работает секвенсор App-v, нажмите кнопку **запустить**  /  **все программы**  /  **Microsoft**Application Virtualization  /  **Sequencer Microsoft**Application Virtualization.

2.  В секвенсоре App-V выберите **изменить существующий виртуальный пакет приложения**, а затем нажмите кнопку **Далее**.

3.  На странице **Выбор задачи** выберите **обновить приложение в существующем пакете**и нажмите кнопку **Далее**.

4.  На странице **выберите пакет** нажмите кнопку **Обзор** , чтобы найти пакет виртуального приложения, содержащий приложение, которое вы хотите обновить, и нажмите кнопку **Далее**.

5.  На странице **Подготовка компьютера** ознакомьтесь с проблемами, которые могут привести к сбою обновления приложения, а также в случае, если обновление приложения содержит ненужные данные. Прежде чем продолжить, настоятельно рекомендуем устранить все возможные проблемы. После устранения конфликтов Обновите отображаемые сведения и нажмите кнопку **Обновить**. После устранения всех возможных проблем нажмите кнопку **Далее**.

    **Важно!**  Если вы хотите отключить антивирусную программу, просканируйте компьютер, на котором запущены программы Sequencer, и убедитесь, что в пакет не добавлены нежелательные или вредоносные файлы.

     

6.  На странице " **Выбор установщика** " нажмите кнопку **Обзор** и укажите файл установки обновления для приложения. Если обновление не имеет связанного файла установщика и вы планируете выполнить все инструкции по установке вручную, установите флажок **выбрать этот параметр, чтобы выполнить выборочную установку** , и нажмите кнопку **Далее**.

7.  На странице **установки** , когда секвенсор и установщик приложения готовы, установите обновление приложения, чтобы секвенсор мог отслеживать процесс установки. Если в процессе установки необходимо запустить дополнительные файлы установки, нажмите кнопку **выполнить** , найдите и запустите дополнительные файлы установки. Когда вы закончите установку, нажмите кнопку **я Заготов установить**. Нажмите кнопку **Далее**.

    **Примечание**  Секвенсор отслеживает все изменения и установки на компьютере, на котором запущены программы Sequencer, включая изменения и установки, выполненные за пределами мастера виртуализации.

     

8.  На странице **отчета об установке** можно просмотреть сведения о виртуальном приложении, которое вы только что обновили. Чтобы получить более подробное описание сведений, отображаемых в **дополнительных сведениях**, дважды щелкните событие. После того как вы проверили данные, нажмите кнопку **Далее**.

9.  На странице **потоковой передачи** запустите каждую программу, чтобы ее можно было оптимизировать и эффективно работать на конечных компьютерах. Выполнение всех приложений может занять несколько минут. После запуска всех приложений закройте все приложения и нажмите кнопку **Далее**.

    **Примечание**  Если вы хотите остановить загрузку приложения на этом этапе, в диалоговом окне **Запуск приложения** нажмите кнопку **остановить**, а затем выберите один из следующих параметров, **Остановите все приложения** или **остановите только это приложение**в зависимости от того, что вы хотите.

     

10. На странице " **Создание пакета** ", чтобы изменить пакет, не сохраняя его, установите флажок **продолжить изменение пакета без сохранения с помощью редактора пакетов** . При выборе этого параметра открывается пакет на консоли секвенсор, и вы можете изменить пакет перед сохранением. Нажмите кнопку **Далее**.

    Чтобы сохранить пакет немедленно, выберите параметр **сохранить пакет**, который будет использоваться по умолчанию. Добавьте необязательные **Примечания** , которые будут связаны с пакетом. Комментарии полезны для идентификации версии и других сведений о пакете. Также будет отображаться **расположение для сохранения** по умолчанию. Чтобы изменить расположение по умолчанию, нажмите кнопку **Обзор** и укажите новое расположение. Отобразится размер пакета без сжатия. Если размер пакета превышает 4 ГБ (несжатые) и вы планируете передавать пакеты на целевые компьютеры, необходимо выбрать команду **сжать пакет**и нажать кнопку **создать**.

11. На странице **завершения** нажмите кнопку **Закрыть** , чтобы закрыть окно мастера. Пакет теперь доступен в секвенсоре.

**Изменение свойств, связанных с существующим пакетом виртуального приложения**

1.  Чтобы запустить секвенсор App-v на компьютере, на котором работает секвенсор App-v, нажмите кнопку **запустить**  /  **все программы**  /  **Microsoft**Application Virtualization  /  **Sequencer Microsoft**Application Virtualization.

2.  В секвенсоре App-V выберите **изменить существующий виртуальный пакет приложения**, а затем нажмите кнопку **Далее**.

3.  На странице **Выбор задачи** выберите **изменить пакет**и нажмите кнопку **Далее**.

4.  На странице **выберите пакет** нажмите кнопку **Обзор** , чтобы найти виртуальный пакет приложения, содержащий свойства приложения, которые вы хотите изменить, и нажмите кнопку **изменить**.

5.  В консоли Sequencer можно выполнить любые из указанных ниже задач.

    -   Просмотр свойств пакета.

    -   Просмотр журнала изменений в пакете.

    -   Просмотр связанных файлов пакетов.

    -   Редактирование параметров реестра.

    -   Ознакомьтесь с дополнительными параметрами пакета (кроме свойств файла операционной системы).

    -   Создайте соответствующий установщик Windows (MSI).

    -   Изменить OSD файл.

    -   Сжатие и распаковка пакета.

    -   Добавление сопоставлений типов файлов.

    -   Задает состояние виртуальной клавиши в реестре (переопределение или слияние).

    -   Настройте состояние виртуализированной папки.

    -   Изменение сопоставлений виртуальной файловой системы.

6.  Завершив изменение свойств пакета, нажмите **файл**  /  **сохранить** , чтобы сохранить пакет.

**Добавление нового приложения в существующий виртуальный пакет приложения**

1.  Чтобы запустить секвенсор App-v на компьютере, на котором работает секвенсор App-v, нажмите кнопку **запустить**  /  **все программы**  /  **Microsoft**Application Virtualization  /  **Sequencer Microsoft**Application Virtualization.

2.  В секвенсоре App-V выберите **изменить существующий виртуальный пакет приложения**, а затем нажмите кнопку **Далее**.

3.  На странице **Выбор задачи** выберите команду **Добавить новое приложение**и нажмите кнопку **Далее**.

4.  На странице **выберите пакет** нажмите кнопку **Обзор** , чтобы найти виртуальный пакет приложений, в который нужно добавить приложение, и нажмите кнопку **Далее**.

5.  На странице **Подготовка компьютера** изучите проблемы, которые могут привести к сбою при создании пакета или наличие ненужных данных в обновлении. Прежде чем продолжить, настоятельно рекомендуем устранить все возможные проблемы. После устранения конфликтов Обновите отображаемые сведения и нажмите кнопку **Обновить**. После устранения всех возможных проблем нажмите кнопку **Далее**.

    **Важно!**  Если вы хотите отключить антивирусную программу, просканируйте компьютер, на котором запущены программы Sequencer, и убедитесь в том, что в пакет нельзя добавить нежелательные или опасные файлы.

     

6.  На странице " **Выбор установщика** " нажмите кнопку **Обзор** и укажите файл установки для приложения. Если у приложения нет связанного файла установщика и вы планируете выполнить все инструкции по установке вручную, установите флажок **выбрать этот параметр, чтобы выполнить выборочную установку** , и нажмите кнопку **Далее**.

7.  На странице **установки** , когда секвенсор и установщик приложения готовы, установите приложение, чтобы секвенсор мог отслеживать процесс установки. Если в процессе установки должны выполняться дополнительные файлы установки, нажмите кнопку **выполнить**, а затем найдите и запустите дополнительные файлы установки. Когда вы закончите установку, нажмите кнопку **я Заготов установить**. Нажмите кнопку **Далее**. В диалоговом окне **Поиск папки** укажите основной каталог, в который будет устанавливаться приложение. Это должно быть новое расположение, чтобы не перезаписать существующую версию виртуального пакета приложений.

    **Примечание**  Все изменения и установки на компьютере, на котором запущен секвенсор, контролируются секвенсором, включая изменения и установки, которые выполняются за пределами мастера виртуализации.

     

8.  На странице **Настройка программного обеспечения** при необходимости запустите программы, содержащиеся в пакете. Этот шаг позволяет выполнить все связанные задачи лицензирования или настройки, необходимые для запуска приложения перед развертыванием и запуском пакета на целевом компьютере. Чтобы запустить сразу все программы, выберите хотя бы одну программу и нажмите кнопку **выполнить все**. Чтобы запустить определенные программы, выберите программу или программы, которые вы хотите запустить, и нажмите кнопку **выполнить выбрано**. Заполните необходимые задачи настройки и закройте приложения. Для запуска всех программ может потребоваться несколько минут. Нажмите кнопку **Далее**.

9.  На странице **отчета об установке** можно просмотреть сведения о виртуальном приложении, которое вы только что обновили. Чтобы получить более подробное описание сведений, отображаемых в **дополнительных сведениях**, дважды щелкните событие. После того как вы проверили данные, нажмите кнопку **Далее**.

10. На странице " **Настройка** ", если вы закончите установку и настройку виртуального приложения, нажмите кнопку **остановить сейчас** и перейдите к шагу 14 этой процедуры. Если вы хотите настроить любой из элементов, указанных в списке ниже, нажмите кнопку **настроить**.

    -   Изменение сопоставлений типов файлов, связанных с приложением.

    -   Подготовьте виртуальный пакет для потоковой передачи. Потоковая передача повышает удобство работы с виртуальным пакетом приложения на целевом компьютере.

    Нажмите кнопку **Далее**.

11. На странице **изменение сочетаний клавиш** можно дополнительно настроить сопоставления типов файлов (FTA), которые будут связаны с различными приложениями в пакете. Чтобы создать новый FTA, в левой области выберите и разверните приложение, которое вы хотите настроить, и нажмите кнопку **Добавить**. В диалоговом окне **Добавление сопоставления типа файлов** укажите необходимые сведения о новом FTA. Чтобы просмотреть сведения о сочетаниях, связанных с приложением, в приложении установите флажок **сочетания клавиш** и в области **Расположение** можно просмотреть сведения о файле значка. Чтобы изменить существующий FTA, нажмите кнопку **изменить**. Чтобы удалить FTA, выберите FTA и нажмите кнопку **Удалить**. Нажмите кнопку **Далее**.

12. На странице **потоковой передачи** запустите каждую программу, чтобы ее можно было оптимизировать и эффективно работать на конечных компьютерах. Выполнение всех приложений может занять несколько минут. После запуска всех приложений закройте все приложения и нажмите кнопку **Далее**.

    **Примечание**  Если вы хотите остановить загрузку приложения на этом этапе, в диалоговом окне **Запуск приложения** нажмите кнопку **остановить** и установите флажок остановить **все приложения** или **остановить только это приложение** , в зависимости от того, что вы хотите.

     

13. На странице " **Создание пакета** " установите флажок **продолжить изменение пакета без сохранения с помощью редактора пакетов** , чтобы изменить пакет, не сохраняя его. При выборе этого параметра открывается пакет на консоли секвенсор, и вы можете изменить пакет перед сохранением. Нажмите кнопку **Далее**.

    Выберите параметр **сохранить пакет**по умолчанию, чтобы сохранить пакет немедленно. Добавьте необязательные **Примечания** , которые будут связаны с пакетом. Комментарии полезны для идентификации версии и других сведений о пакете. Также будет отображаться **расположение для сохранения** по умолчанию. Чтобы изменить расположение по умолчанию, нажмите кнопку **Обзор** и укажите новое расположение. Отобразится размер пакета без сжатия. Если размер пакета превышает 4 ГБ (без сжатия) и вы планируете потоковую передачу пакета на целевые компьютеры, необходимо выбрать команду **сжать пакет**. Нажмите кнопку **Создать**.

14. На странице **Завершение** щелкните **Закрыть**. Пакет теперь доступен в секвенсоре.
