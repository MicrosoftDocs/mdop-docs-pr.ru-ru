<span data-ttu-id="94eeb-104">Последовательность, процесс, используемый Application Virtualization для создания виртуальных приложений и пакетов приложений, требует использования компьютера с установленным программным обеспечением Application Virtualization Sequencer.</span><span class="sxs-lookup"><span data-stu-id="94eeb-104">Sequencing, the process used by Application Virtualization to create virtual applications and application packages, requires the use of a computer with the Application Virtualization Sequencer software installed.</span></span>

<span data-ttu-id="94eeb-105">В процессе упорядочивания секвенсор помещается в режим монитора, и приложение для упорядочения установлено на компьютере виртуализации.</span><span class="sxs-lookup"><span data-stu-id="94eeb-105">During the sequencing process, the Sequencer is placed in monitor mode, and the application to be sequenced is installed on the sequencing computer.</span></span> <span data-ttu-id="94eeb-106">Затем запускается упорядоченное приложение, и наиболее важные и часто используемые функции выполняются, чтобы процесс мониторинга мог настроить основной блок функций, который содержит минимальное содержимое пакета приложения, которое необходимо для запуска приложения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-106">Next, the sequenced application is started, and its most important and commonly used functions are exercised so that the monitoring process can configure the primary feature block, which contains the minimum content in an application package that is necessary for an application to run.</span></span> <span data-ttu-id="94eeb-107">После выполнения этих действий режим наблюдения останавливается, а приложение, упорядоченное для проверки правильности, будет сохранено и протестировано.</span><span class="sxs-lookup"><span data-stu-id="94eeb-107">When these steps are complete, monitoring mode is stopped and the sequenced application is saved and tested to verify correct operation.</span></span>

<span data-ttu-id="94eeb-108">При принятии решения о том, какие приложения нужно выбрать для упорядочивания, следует помнить, что некоторые приложения не могут быть упорядочены.</span><span class="sxs-lookup"><span data-stu-id="94eeb-108">When deciding which applications to choose for sequencing, remember that certain applications cannot be sequenced.</span></span> <span data-ttu-id="94eeb-109">Сюда входят некоторые компоненты операционной системы Windows, например Internet Explorer, драйверы устройств и приложения, запускающие службы во время загрузки.</span><span class="sxs-lookup"><span data-stu-id="94eeb-109">These include certain parts of the Windows operating system, such as Internet Explorer, device drivers, and applications that start services at boot time.</span></span>

<span data-ttu-id="94eeb-110">Пошаговые инструкции по установке секвенсора вы узнаете, [как установить Application Virtualization Sequencer](how-to-install-the-application-virtualization-sequencer.md).</span><span class="sxs-lookup"><span data-stu-id="94eeb-110">For step-by-step information about installing the Sequencer, see [How to Install the Application Virtualization Sequencer](how-to-install-the-application-virtualization-sequencer.md).</span></span>

<span data-ttu-id="94eeb-111">**Важно!**  Весь план обработки последовательности должен быть проверен и утвержден группой корпоративной безопасности.</span><span class="sxs-lookup"><span production environment in a lab.</span></span> <span data-ttu-id="94eeb-113">Это может быть как простым, так и более полным, в зависимости от требований бизнеса.</span><span class="sxs-lookup"><span подключены к корпоративной сети, чтобы копировать готовые пакеты на производственные серверы.</span><span class="sxs-lookup"><span data-stu-id="94eeb-114">The sequencing computers will need connectivity to the corporate network to copy finished packages over to the production servers.</span></span> <span data-ttu-id="94eeb-115">Тем не менее, так как они обычно работают без защиты от behind a firewall or on an isolated network segment.</span></span> <span data-ttu-id="94eeb-116">Использование виртуальных машин, настроенных для совместного использования изолированной виртуальной сети, также может быть приемлемым подходом.</span><span class="sxs-lookup"><span data-stu-id="94eeb-116">Using Virtual Machines configured to share an isolated virtual network might also be an <span data-ttu-id="94eeb-118">Ниже приведены ключевые действия по планированию процесса упорядочения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-118">Key steps for planning the sequencing process include the following:</span></span>

-   <span data-ttu-id="94eeb-119">Определите количество заявлений, которые вы планируете обрабатывать ежемесячно, размер этих приложений и добавьте скидку для очередности будущих обновлений.</span><span class="sxs-lookup"><span data-stu-id="94eeb-119">Consider the number of applications you expect to process each month, the size of those applications, and add an allowance for sequencing future updates.</span></span> <span data-ttu-id="94eeb-120">Пакеты могут иметь размер до 4 ГБ, сжатый или несжатой.</span><span class="sxs-lookup"><span data-stu-id="94eeb-120">Packages can be up to 4 GB in size, compressed or uncompressed.</span></span>

-   <span data-ttu-id="94eeb-121">Подготовьте и задокументируйте methodical, повторяемый процесс для вашей организации, чтобы подписаться на них при виртуализации каждого приложения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-121">Prepare and document a methodical, repeatable process for your organization to follow when sequencing each application.</span></span> <span data-ttu-id="94eeb-122">Это должно включать использование контрольного списка для каждого выполнения, а также процесс управления версиями.</span><span class="sxs-lookup"><span data-stu-id="94eeb-122">This should include the use of a checklist for each run, as well as a version control process.</span></span> <span data-ttu-id="94eeb-123">Использование журнала отслеживания для каждого последовательного приложения также очень полезно при исследовании возможных технических проблем с пакетом.</span><span class="sxs-lookup"><span data-stu-id="94eeb-123">The use of a tracking log for each sequenced application is also very helpful when investigating possible technical issues with a package.</span></span>

-   <span data-ttu-id="94eeb-124">Для виртуализации приложений используйте высокопроизводительные компьютеры, оптимизированные для обработки пропускной способности (не менее 4 ГБ оперативной памяти и быстрый ЦП (3 ГГц или выше).</span><span class="sxs-lookup"><span data-stu-id="94eeb-124">For sequencing applications, use high-performing computers that are optimized for processing throughput, with at least 4 GB of RAM and a fast CPU (3 GHz or faster).</span></span> <span data-ttu-id="94eeb-125">Кроме того, вы можете повысить производительность с помощью быстрых жестких дисков и использования отдельных дисковых томов.</span><span class="sxs-lookup"><span data-stu-id="94eeb-125">Fast hard disks and the use of separate disk volumes can also improve performance.</span></span> <span data-ttu-id="94eeb-126">Виртуальные машины идеально подходят для упорядочивания, так как они могут быть легко сброшены, или можно использовать физический компьютер с чистым изображением в локальной секции для быстрой повторной обработки изображений после завершения каждой операции последовательности пакетов.</span><span class="sxs-lookup"><span data-stu-id="94eeb-126">Virtual Machines are ideal for sequencing because they can easily be reset, or you can use a physical computer with a clean image on a local partition to enable rapid re-imaging after each package sequencing operation has been completed.</span></span>

    <span data-ttu-id="94eeb-127">**Важно!**  Выполнение секвенсора App-V в безопасном режиме не поддерживается.</span><span class="sxs-lookup"><span data-stu-id="94eeb-127">**Important** Running the App-V sequencer in Safe Mode is not supported.</span></span>

     

-   <span data-ttu-id="94eeb-128">Убедитесь, что вы понимаете операционную среду упорядоченного приложения, включая элементы интеграции, такие как Microsoft Office или среда выполнения Java, так как это часто определяет необходимость установки каких-либо элементов на компьютере виртуализации перед виртуализацией приложения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-128">Verify that you understand the sequenced application’s operating environment, including integration elements such as Microsoft Office or the Java Runtime Environment, because this will often determine whether anything has to be installed on the sequencing computer prior to sequencing the application.</span></span>

-   <span data-ttu-id="94eeb-129">Убедитесь, что каждая новая операция последовательности всегда начинается с чистого базового образа.</span><span class="sxs-lookup"><span data-stu-id="94eeb-129">Ensure that each new sequencing operation always starts with a clean base image.</span></span> <span data-ttu-id="94eeb-130">Убедитесь, что на компьютере, на котором установлен виртуализация, Восстановление сохраненного образа на физическом компьютере или перезапуск виртуальной машины после отмены всех изменений.</span><span class="sxs-lookup"><span data-stu-id="94eeb-130">Make sure that the sequencing computer has been reset, either by restoring the saved image to a physical computer or by restarting a virtual machine after discarding all changes.</span></span> <span data-ttu-id="94eeb-131">Перед сохранением базового образа должны быть установлены последние обновления из центра обновления Windows.</span><span class="sxs-lookup"><span data-stu-id="94eeb-131">The base image should have the latest updates applied from Windows Update before saving.</span></span>

-   <span data-ttu-id="94eeb-132">Вы можете отключить любые элементы на компьютере виртуализации, которые могут помешать процессу наблюдения за установкой, таким как антивирусные сканеры и веб-центра обновления Windows, так как при этом необходимо иметь стабильную платформу во время процесса упорядочения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-132">Turn off anything on the sequencing computer that can interfere with the install monitoring process, such antivirus scanners and Windows Update, because having a stable platform during the sequencing process is essential.</span></span> <span data-ttu-id="94eeb-133">Поскольку это приводит к значительным угрозам безопасности, убедитесь в том, что предприняты необходимые меры для защиты компьютера и сети, а также серийного пакета приложения.</span><span class="sxs-lookup"><span data-stu-id="94eeb-133">Because this step incurs significant security risks, ensure that the correct precautions are taken to protect the computer and network as well as the sequenced application package.</span></span> <span data-ttu-id="94eeb-134">Рекомендуется проверять пакеты приложения антивирусной программой перед их упорядочением.</span><span class="sxs-lookup"><span data-stu-id="94eeb-134">We recommend that you do an antivirus scan of application packages before sequencing them.</span></span>

-   <span data-ttu-id="94eeb-135">Включите подробный процесс тестирования каждого приложения после выполнения последовательности.</span><span class="sxs-lookup"><span data-stu-id="94eeb-135">Include a detailed process for testing each application after sequencing.</span></span> <span data-ttu-id="94eeb-136">Тестирование приложения, упорядоченного по порядку, определяет, правильно ли оно работает и является важной частью процесса перед развертыванием виртуализованного приложения для конечных пользователей.</span><span class="sxs-lookup"><span data-stu-id="94eeb-136">Testing the sequenced application will determine whether it functions correctly and is an essential part of the process prior to deploying the virtualized application to end users.</span></span> <span data-ttu-id="94eeb-137">В качестве заключительного шага при тестировании перед развертыванием в масштабах до широкой шкалы конечные пользователи также следует планировать пилотное развертывание для тестовой группы.</span><span class="sxs-lookup"><span data-stu-id="94eeb-137">As the final step in testing prior to wide-scale deployment to end users, you should also plan for a pilot deployment to a test group.</span></span>

-   <span data-ttu-id="94eeb-138">При тестировании последовательных приложений выберите компьютерное оборудование того же типа и запустите те же операционные системы, которые используются в производственной среде компании.</span><span class="sxs-lookup"><span data-stu-id="94eeb-138">When testing sequenced applications, choose computer equipment of the same type and running the same operating systems that are in use in the company production environment.</span></span> <span data-ttu-id="94eeb-139">Если они настроены правильно, можно использовать виртуальные машины или физические компьютеры.</span><span class="sxs-lookup"><span data-stu-id="94eeb-139">As long as they are configured properly, either virtual machines or physical machines can be used.</span></span>
