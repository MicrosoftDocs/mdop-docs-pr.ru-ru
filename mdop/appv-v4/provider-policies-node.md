# Узел политик поставщика


Узел **политики поставщика** — это один из уровней ниже узла системы Application Virtualization в области **область** консоли управления сервером виртуализации приложений. При выборе этого узла в области **результатов** отображается список политик поставщика. Щелкните правой кнопкой мыши узел **политики поставщика** , чтобы открыть всплывающее меню, содержащее указанные ниже элементы.

<a href="" id="new-provider-policy"></a>**Новая политика поставщика**  
Отображение мастера создания политики поставщика. Этот мастер состоит из следующих страниц:

1.  Введите имя в поле **имя политики поставщика** . Выберите " **Управление рабочим столом на клиентском компьютере" с помощью консоли управления** , если эта возможность нужна. Установите один или оба указанных ниже флажка, если вы хотите применить соответствующие функции.

    -   **Обновление конфигурации публикации при входе пользователя**

    -   **Обновлять конфигурацию каждые**. После выбора этого параметра введите номер и выберите единицу в раскрывающемся меню. Диапазон допустимых записей не должен превышать **30 минут** в течение не менее **999 дней**.

2.  Нажмите кнопку **Добавить** или **Удалить** , чтобы добавить или удалить назначение группы. Используйте стандартное диалоговое окно " **Обзор** " для поиска группы пользователей.

3.  Установите один из указанных ниже флажков в диалоговом окне **Настройка конвейера поставщика** , чтобы включить соответствующую функцию.

    -   **Проверка подлинности**— выберите тип проверки подлинности из раскрывающегося списка.

    -   **Принудительная установка разрешений на доступ**

    -   **Ведение журнала сведений об использовании**

    -   **Лицензирование**: выберите схему принудительного применения из раскрывающегося списка.

4.  Нажмите кнопку **Готово** , чтобы добавить новую политику поставщика.

<a href="" id="view"></a>**Просмотр**  
Изменение внешнего вида и содержимого области **результатов** .

<a href="" id="new-window-from-here"></a>**Новое окно**  
Открытие новой консоли управления с выбранным узлом в качестве корневого узла.

<a href="" id="refresh"></a>**Обновить**  
Обновляет представление сервера.

<a href="" id="export-list"></a>**Экспорт списка**  
Создает текстовый файл с разделителями-знаками табуляции, который включает содержимое области **результатов** . Этот элемент отображает диалоговое окно " **Сохранение файла** ", в котором указывается расположение для текстового файла, который вы создаете.

<a href="" id="help"></a>**Help**  
Вывод справочной системы для консоли управления сервером виртуализации приложений.
