# Печать отчета


Процесс печати отчета для консоли управления сервером Application Virtualization одинаков, независимо от типа отчета.

**Печать отчета**

1.  После отображения вывода отчета в области **результатов** нажмите кнопку **принтер** (в верхней части области **результатов** ) или щелкните правой кнопкой мыши нужный отчет в разделе " **отчеты** " и выберите команду " **Печать** " в контекстном меню.

2.  Следуйте инструкциям по печати и нажмите кнопку **ОК**.
