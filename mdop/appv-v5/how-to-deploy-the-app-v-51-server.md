# Порядок развертывания сервера App-V 5.1


Чтобы установить сервер Microsoft Application Virtualization (App-V) 5,1, выполните указанные ниже действия. Сведения о развертывании сервера App-V 5,1 можно найти в разделе [о приложении App-v 5,1](about-app-v-51.md#bkmk-migrate-to-51).

**Прежде чем начать, выполните указанные ниже действия.**

-   Убедитесь, что вы установили необходимое программное обеспечение. Ознакомьтесь [с требованиями для App-V 5,1](app-v-51-prerequisites.md).

-   Ознакомьтесь с разделом "сервер" в разделе "безопасность" для [App-V 5,1](app-v-51-security-considerations.md).

-   Укажите порт, на котором будет размещен каждый из компонентов.

-   Добавьте правила брандмауэра, разрешающие входящим запросам доступ к указанным портам.

-   Если вы используете скрипты SQL вместо установщика Windows, чтобы настроить базу данных управления или базу данных отчетов, необходимо выполнить сценарии SQL перед установкой сервера управления или сервера отчетов. Узнайте, [как развертывать базы данных App-V с помощью сценариев SQL](how-to-deploy-the-app-v-databases-by-using-sql-scripts51.md).

**Установка сервера App-V 5,1**

1. Скопируйте файлы установки сервера App-V 5,1 на компьютер, на котором вы хотите установить его.

2. Запустите установку сервера App-V 5,1, щелкнув правой кнопкой мыши **appv\_server\_setup.exe** с помощью учетной записи администратора, а затем выберите команду **установить**.

3. Проверьте и приняв условия лицензионного соглашения и выберите, нужно ли включать обновления Майкрософт.

4. На странице **Выбор компонентов** выберите все следующие компоненты.
