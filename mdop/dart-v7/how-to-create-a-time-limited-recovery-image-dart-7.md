# Создание образа для восстановления с ограничением по времени


Вы можете создать изображение для восстановления DaRT, которое может быть использовано только в течение определенного количества дней после его создания. Для этого необходимо запустить **Мастер создания образа для восстановления DaRT** в командной строке и указать количество дней.

**Создание образа восстановления с ограничением по времени**

1.  Откройте командную строка с учетными данными администратора.

2.  Перейдите в папку, в которой находится программа ERDC.exe.

3.  С помощью приведенного ниже синтаксиса запустите **Мастер создания образа для восстановления DaRT**. *NumberOfDays* — это положительное целое число, представляющее количество дней, в течение которых будет использоваться изображение для восстановления DART.

    ``` syntax
    ERDC /e NumberOfDays
    ```
