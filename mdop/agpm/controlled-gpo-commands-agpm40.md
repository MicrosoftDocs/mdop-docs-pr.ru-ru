# Команды управляемого объекта групповой политики


Вкладка **Управление** :

-   Отображение списка объектов групповой политики (GPO), управляемых с помощью расширенного управления групповыми политиками.

-   Контекстное меню с командами для управления объектами групповой политики и отображения журнала и отчетов для GPO.

-   Выводит список групп и пользователей, которые имеют разрешение на доступ к выбранному объекту групповой политики.

Если щелкнуть правой кнопкой мыши список **объектов групповой политики** на этой вкладке, отобразится контекстное меню. Это меню содержит любые из указанных ниже вариантов.

Если у вас нет разрешения на развертывание объекта групповой политики, вам будет предложено отправить запрос.</p></td>
</tr>
<tr class="odd">
<td align="left"><p><strong>Экспорт в</strong></p></td>

</tr>
<tr class="even">
<td align="left"><p><strong>Label</strong></p></td>
<td align="left"><p>Пометка выбранного объекта групповой политики с описательной надписью (например, &quot; известная хорошая &quot; ) и примечания для сохранения записи. Метки отображаются в <strong> столбце "состояние" </strong> и примечания в <strong> столбце "Примечание" в </strong> <strong> окне "журнал" </strong> .
