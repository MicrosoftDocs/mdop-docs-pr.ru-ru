# Настройка опубликованных приложений


Приложения, несовместимые с операционной системой узла, можно запускать в рабочей области MED-V и инициировать из рабочей области MED-V таким же образом, как и на рабочем столе — в меню "Пуск" или с помощью ярлыка на локальном узле. Выделены и определены приложения, которые называются опубликованными приложениями. Процедуры, описанные в этом разделе, описывают, как добавлять и удалять опубликованные приложения.

Приложение может быть опубликовано одним из указанных ниже способов.

-   Как приложение — выберите определенное приложение, введя его в командную строку приложения. Опубликовано только выбранное приложение.

-   Меню — выберите папку, которая включает несколько приложений. Все приложения в папке публикуются и отображаются в виде меню.

## <a href="" id="bkmk-addingapublishedapplication"></a>Добавление опубликованного приложения в рабочую область MED-V


**Добавление приложения в рабочую область MED-V**

1.  Щелкните рабочую область MED-V, которую нужно настроить.

2.  В области **приложения** в разделе **опубликованные приложения** нажмите кнопку **добавить** , чтобы добавить новое приложение.

3.  Настройте свойства приложения, как описано в приведенной ниже таблице.

4.  В меню **Политика** выберите команду **сохранить**.

    **Примечание.**  
    Если вы настраиваете Internet Explorer как опубликованное приложение, чтобы обеспечить правильную работу перенаправления веб-браузера, убедитесь в том, что параметры не заключены в круглые скобки.

## Удаление опубликованного приложения из рабочей области MED-V


**Удаление приложения из рабочей области MED-V**

1.  Щелкните рабочую область MED-V.

2.  В области **приложения** в разделе **опубликованные приложения** выберите приложение, которое нужно удалить.

3.  Нажмите кнопку **Удалить**.

    Приложение удаляется из списка опубликованных приложений.

4.  В меню **Политика** выберите команду **сохранить**.

## Удаление опубликованного меню из рабочей области MED-V


**Удаление опубликованного меню из рабочей области MED-V**

1.  Щелкните рабочую область MED-V.

2.  На панели **приложения** в разделе **опубликованные меню** выберите меню, которое нужно удалить.

3.  Нажмите кнопку **Удалить**.

    Меню удаляется из списка опубликованных меню.

4.  В меню **Политика** выберите команду **сохранить**.

## Запуск опубликованного приложения из командной строки на клиентском компьютере


Администратор может запускать опубликованные приложения из любого места, такого как ярлык на рабочем столе, с помощью следующей команды:

``` syntax
"<Install path>\Manager\KidaroCommands.exe" /run "<published application name>" "<MED-V workspace name>"
```

**Примечание.**  
Рабочая область MED-V, в которой определено опубликованное приложение, должна выполняться.
