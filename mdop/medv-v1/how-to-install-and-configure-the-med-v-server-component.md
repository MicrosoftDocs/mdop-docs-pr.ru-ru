# Установка и настройка серверного компонента MED-V Server


В этом разделе объясняется, как [установить](#bkmk-howtoinstallthemedvserver) и [настроить](#bkmk-howtoconfigurethemedvserver) сервер MED-V.

## <a href="" id="bkmk-howtoinstallthemedvserver"></a>Установка сервера MED-V


**Установка сервера MED-V**

1.  Установите пакет приложения MED-V Server. msi.

    Пакет для MED-V Server. msi называется *MED-V\_Server\_x.msi*, где x — номер версии.

    Например, *MED-V\_Server\_1.0.65.msi*.

2.  После появления экрана **приветствия мастера InstallShield** нажмите кнопку **Далее**.

3.  На экране лицензионного **соглашения** ознакомьтесь с лицензионным соглашением, выберите **я принимаю условия лицензионного соглашения**и нажмите кнопку **Далее**.

    Откроется экран **Конечная папка** с отображаемой папкой "Установка по умолчанию".

    По умолчанию используется папка для установки *%systemdrive%\\Program Files\\Microsoft Enterprise настольная Virtualization\\*.

    -   Чтобы изменить папку, в которой должна быть установлена версия MED-V, нажмите кнопку **изменить** и перейдите к существующей папке.

4.  Нажмите кнопку **Далее**.

5.  На экране **Готово для установки программы** нажмите кнопку **установить**.

    Запустится установка сервера MED-V. Это может занять несколько минут, и на экране может не отображаться текст. Во время установки появляются несколько экранов хода выполнения. Если появится сообщение, следуйте инструкциям, приведенным в этой области.

6.  После появления экрана **мастера InstallShield** нажмите кнопку **Готово** , чтобы завершить работу мастера.

**Примечание.**  
Если вы устанавливаете сервер MED-V через Microsoft Remote Desktop, используйте следующий синтаксис: **mstsc или Admin**. Убедитесь, что сеанс подключения к удаленному рабочему столу направлен на консоль.



## <a href="" id="bkmk-howtoconfigurethemedvserver"></a>Настройка сервера MED-V


Можно настроить следующие параметры сервера:

-   [Подключения](#bkmk-configuringconnections)

-   [Изображения](#bkmk-configuringimages)

-   [Разрешения](#bkmk-configuringpermissions)

-   [Отчеты](#bkmk-configuringreports)

### <a href="" id="bkmk-configuringconnections"></a>Настройка подключений

**Настройка подключений**

1.  В меню Пуск Windows выберите **все программы, &gt; &gt; Диспетчер конфигурации сервера**med-v.

    **Примечание.**  
    Примечание. Если при установке сервера установлен флажок **запустить диспетчер конфигурации MED-v Server** , диспетчер конфигурации MED-v автоматически запускается после завершения установки.



~~~
The MED-V Server Configuration Manager appears.
~~~

2. На вкладке **подключения** настройте следующие параметры клиентских подключений:

   -   **Включить нешифрованные соединения (http) с помощью порта**— установите этот флажок, чтобы включить незашифрованные подключения с помощью указанного порта. В поле Port (порт) введите порт сервера, на котором вы хотите использовать незашифрованные соединения (http).

   -   **Включить шифрование подключений (HTTPS) с помощью порта**— установите этот флажок, чтобы включить шифрование подключений с помощью указанного порта. В поле Port (порт) введите порт сервера, на котором вы хотите использовать зашифрованные подключения (HTTPS).

       HTTPS — это необязательная конфигурация, с помощью которой можно настроить безопасные транзакции между сервером MED-V и клиентами MED-V. Для настройки HTTPS необходимо выполнить описанные ниже действия.

       -   Настройка сертификата на сервере.

       -   Свяжите сертификат сервера с портом, указанным с помощью команды Netsh. Дополнительные сведения можно найти в перечисленных ниже случаях.

           -   [Команды Netsh для протоколов передачи гипертекста (HTTP)](https://go.microsoft.com/fwlink/?LinkId=183314)

           -   [Инструкции: Настройка порта с помощью SSL-сертификата](https://go.microsoft.com/fwlink/?LinkID=183315)

           -   [Инструкции: Настройка порта с помощью SSL-сертификата](https://msdn.microsoft.com/library/ms733791.aspx)

3. Нажмите кнопку **ОК**.

### <a href="" id="bkmk-configuringimages"></a>Настройка изображений

**Настройка изображений**

1.  Откройте вкладку **изображения** .

2.  Настройте следующие параметры управления образами:

    -   **Каталог ВМ**— каталог виртуальной машины (каталог, в котором хранятся изображения). Это поле имеет UNC-путь к каталогу изображений на сервере распространения изображений, который должен быть доступен на сервере MED-V.

    -   **URL-адрес ВМ**— расположение сервера, на котором хранятся изображения.

3.  Нажмите кнопку **ОК**.

### <a href="" id="bkmk-configuringpermissions"></a>Настройка разрешений

**Настройка разрешений**

1.  Откройте вкладку **разрешения** .

2.  Предоставляется список всех пользователей, которые могут войти в систему. Чтобы применить разрешения на чтение и запись для пользователя, установите флажок рядом с нужным пользователем. Чтобы применить к пользователю разрешения только для чтения, снимите флажок.

3.  Чтобы добавить пользователей домена или группы, нажмите кнопку **Добавить**.

    Откроется диалоговое окно " **Введите имена пользователей или групп** ".

    1.  Выберите пункт Пользователи домена или группы, выполнив одно из указанных ниже действий.

        -   В поле **Введите имена пользователей или групп** введите пользователя или группу, которые есть в домене или находятся в качестве локальных пользователей или групп на компьютере. Затем нажмите кнопку **Проверить имена** , чтобы сопоставить их с полностью существующим именем.

        -   Нажмите кнопку **найти** , чтобы открыть диалоговое окно Стандартный **Выбор пользователей или групп** . Затем выберите пункт Пользователи домена или группы.

    2.  Нажмите кнопку **ОК**.

4.  Для удаления пользователей или групп домена выберите пользователя или группу и нажмите кнопку **Удалить**.

5.  Нажмите кнопку **ОК**.

### <a href="" id="bkmk-configuringreports"></a>Настройка отчетов

**Настройка отчетов**

1.  Откройте вкладку **отчеты** .

2.  Для поддержки отчетов выберите **включить отчеты**.

3.  В поле **строка подключения** введите строку подключения для базы данных MSSQL.

    -   При установке SQL Server на удаленном сервере используйте следующую строку подключения:

        `Data Source=<ServerName>;Initial Catalog=<DBName>;uid=sa;pwd=<Password>;`

        **Примечание.**  
        Примечание. чтобы подключиться к SQL Express, используйте: `Data Source=<ServerName>\sqlexpress.`



4.  Чтобы создать базу данных, нажмите кнопку **создать базу данных**.

5.  Чтобы проверить подключение, нажмите кнопку **проверить соединение**.

6.  Чтобы настроить параметры очистки базы данных, нажмите кнопку **очистить параметры**.

    Откроется диалоговое окно " **Очистка параметров базы данных** ".

    1.  Выберите один из следующих вариантов:

        -   **Очистка данных старше**, очистка всех данных, которые старше указанного количества дней; в поле число введите количество дней.

        -   **Очистите все данные из базы данных**— очистите все данные в базе данных.

        -   **DROP DATABASE**(удалить базу данных) — удаление базы данных.

    2.  Нажмите кнопку **ОК** , чтобы применить изменения и закрыть диалоговое окно.

7.  Нажмите кнопку **ОК** , чтобы сохранить изменения, или кнопку **Отмена** , чтобы закрыть диалоговое окно без сохранения изменений.

8.  Если будет предложено, перезапустите службу сервера MED-V, чтобы применить изменения к сетевым параметрам.
